函数的反汇编分析-嵌套调用

要求：编写一个函数，能够实现对任意5个整数实现加法(使用Plus1和Plus2).

c代码

#include "stdafx.h"

int Plus1(int x,int y)
{
    return x+y;
}
int Plus2(int x,int y,int z)
{
    return x+y+z;
}
int Plus3(int a,int b,int c,int d,int e)
{
    int i,j,m;
    i=Plus1(a,b);
    j=Plus2(c,d,e);
    m=Plus1(i,j);
    return m;
}
int main(int argc, char* argv[])
{
    Plus3(2,1,3,1,5);
    return 0;
}
